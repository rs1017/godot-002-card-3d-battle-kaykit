# Card Selection Rules

## 기본 조작
- 좌클릭: 손패 카드 선택
- 드래그 후 보드 중앙(플레이 영역)에 드롭: 카드 사용 시도
- 우클릭: 현재 선택/타겟팅 취소
- `E`: 턴 종료

## 멀티(2~4인) 턴 구조
- 메인 메뉴에서 플레이어 수(2/3/4) 선택 가능
- 4인 모드(2v2)에서는 Seat 순서로 턴이 순환
- HUD Turn 라벨에 현재 Seat 번호가 표시됨
- 현재 1차 구현은 팀 공유 상태(아군 2명이 동일 손패/마나 풀 공유)

## 카드가 선택/사용되지 않는 경우
- 내 턴이 아님
- 애니메이션/AI 턴 진행 중
- 마나 부족
- 미니언 보드 슬롯 초과(최대 7)

## 공격 규칙
- 내 미니언 클릭 후 타겟 지정
- 적 필드에 `Taunt`가 있으면 해당 미니언만 공격 가능
- `Rush` 미니언은 소환 턴에도 즉시 공격 가능
- `Battlecry`는 카드 사용 시 즉시 발동

## QA 단축키
- `F8`: 키워드(Taunt/Rush/Battlecry) 자동 점검
- `F9`: AI 턴 자동 점검
