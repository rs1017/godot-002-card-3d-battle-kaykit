# Ralph Artifact Guide

이 폴더는 Ralph 반복 사이클 실행 결과를 목적별로 분리해 보관합니다.

## Local Subfolder Guide
- `plans`: 사이클별 계획 문서(`plan-*.md`)
- `runs`: 실행 리포트(`run-*.md`)
- `cycles`: 사이클 검증 요약/루프 요약
- `reviews`: 리뷰 에이전트 결과
- `planning`: 마스터 설계서와 밸런스 CSV
- `reference`: 레퍼런스 수집 이미지 및 메모
- `distribution`: 최신 배포용 계획 문서
- `research`: 조사성/감사성 문서 보관
